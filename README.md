# Kerbal Community

Kerbal Communty är en Kerbal space program fan sida som låter användare läsa om Kerbin, Jebediah Kerman och chatta med andra fans i sidans livechatt. Dessutom finns en webbshopp där användare kan köpa merch.

## Varför finns projektet?
Projektet existerar för att uppfylla uppgiften av en slutuppgift inom kurserna Webbutveckling och Webbserverprogrammering. Dessutom finns projektet som en eventuell grund inför en riktig kerbal space program sida om utvecklingen någon gång skulle önskas.

## Kom igång
Importera kerbal.sql in i phpMyAdmin
